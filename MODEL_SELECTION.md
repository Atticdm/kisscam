# Выбор модели Grok API для Kisscam Bot

## Анализ доступных моделей

### Требования бота:
1. **Анализ изображений** - определение количества людей на фото
2. **Генерация видео** - создание видео с целующимися людьми

### Доступные модели Grok API:

#### Для анализа изображений:
- ✅ **grok-2-vision-1212** - единственная модель с поддержкой Image input
  - Modalities: Text + Image → Text
  - Context: 32,768 токенов
  - Rate limits: 600 rpm
  - Pricing: $2.00 - $10.00 за миллион токенов
  - **Использование**: Определение количества людей на фото

#### Для генерации видео:
- ❌ **Проблема**: В списке доступных моделей НЕТ моделей, которые генерируют видео!
  - Все модели либо Text → Text, либо Text → Image
  - Нет Text/Image → Video моделей

## Выбранная модель

### grok-2-vision-1212

**Использование:**
- ✅ Анализ изображений (определение количества людей)
- ✅ Описание изображений
- ❌ Генерация видео (не поддерживается)

**Почему эта модель:**
1. Единственная модель с поддержкой Image input
2. Может анализировать фотографии
3. Может описывать содержимое изображений
4. Подходит для задачи определения людей на фото

## Проблема: Генерация видео

**Grok API не поддерживает генерацию видео напрямую!**

### Возможные решения:

#### Вариант 1: Использовать другой сервис для видео
- **RunwayML** - генерация видео из изображений
- **Pika Labs** - анимация изображений
- **Stable Video Diffusion** - генерация видео
- **LeiaPix** - создание 3D анимаций

#### Вариант 2: Комбинированный подход
1. Использовать **grok-2-vision-1212** для анализа изображений
2. Использовать **grok-2-image-1212** для генерации промежуточных кадров
3. Собрать кадры в видео (сложно и не оптимально)

#### Вариант 3: Изменить концепцию бота
- Генерировать анимированные GIF вместо видео
- Использовать grok-2-image-1212 для создания нескольких кадров
- Объединить кадры в GIF

## Рекомендация

**Текущая реализация:**
- Используем **grok-2-vision-1212** для анализа изображений ✅
- Для генерации видео нужно интегрировать другой сервис (RunwayML, Pika Labs и т.д.)

**Следующие шаги:**
1. Протестировать grok-2-vision-1212 для определения людей
2. Изучить API RunwayML или Pika Labs для генерации видео
3. Интегрировать выбранный сервис для генерации видео

## Альтернативные модели (если нужны)

### Для текстовых задач:
- **grok-4-fast-non-reasoning** - быстрая модель для простых задач
- **grok-3-mini** - легкая модель для базовых задач

### Для генерации изображений:
- **grok-2-image-1212** - генерация изображений из текста
  - Но не подходит для видео!
