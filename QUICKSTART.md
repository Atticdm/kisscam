# Быстрый старт: Kisscam Bot

## Шаг 1: Установка зависимостей

```bash
# Создайте виртуальное окружение
python3 -m venv venv

# Активируйте его
source venv/bin/activate  # macOS/Linux
# или
venv\Scripts\activate  # Windows

# Установите зависимости
pip install -r requirements.txt
```

## Шаг 2: Проверка конфигурации

Токены нужно настроить в `.env` файле:
- Telegram Bot Token: получите у [@BotFather](https://t.me/BotFather)
- Grok API Key: получите на [x.ai](https://x.ai)

Если нужно изменить, отредактируйте `.env` файл.

## Шаг 3: Запуск бота

```bash
python bot/main.py
```

Вы должны увидеть:
```
INFO     kisscam Bot starting...
```

## Шаг 4: Тестирование

1. Откройте Telegram
2. Найдите вашего бота (username из BotFather)
3. Отправьте команду `/start`
4. Отправьте фотографию с людьми
5. Дождитесь обработки и получите видео!

## Режим двух фотографий

1. Отправьте команду `/two` или текст "Две фотографии"
2. Отправьте первую фотографию
3. Отправьте вторую фотографию
4. Получите видео!

## Устранение проблем

### Бот не запускается
- Проверьте, что виртуальное окружение активировано
- Проверьте, что все зависимости установлены: `pip list`
- Проверьте токены в `.env` файле

### Ошибка при обработке фотографий
- Проверьте логи в `logs/bot.log`
- Убедитесь, что Grok API доступен
- Проверьте формат фотографии (JPG, PNG)

### Grok API ошибки
- Проверьте API ключ
- Убедитесь, что у вас есть доступ к Grok API
- Проверьте лимиты API

## Структура проекта

```
kisscam/
├── bot/              # Код бота
│   ├── main.py       # Точка входа
│   ├── config.py     # Конфигурация
│   └── handlers/     # Обработчики команд и фото
├── services/         # Сервисы (Grok, Image)
├── utils/            # Утилиты (логирование)
├── storage/          # Временное хранилище
└── logs/             # Логи
```
